# SQL & Python Docs

Boshlang'ich darajadagi SQL va Python dasturchilar uchun to'liq dokumentatsiya va o'quv materiallari.

## 📚 Maqsad

Bu repositoriy SQL va Python dasturlash tillarini o'rganayotgan boshlang'ich darajadagi dasturchilar uchun to'liq qo'llanma bo'lib xizmat qiladi.

## 🎯 Mavzular

### Python

- Asosiy sintaksis va ma'lumot turları
- O'zgaruvchilar va data strukturalari
- Funksiyalar va modullar
- Object-Oriented Programming (OOP)
- Fayllar bilan ishlash
- Debugging va xatolik qo'llanish

### SQL

- SELECT asosiy buyruqlari
- WHERE, JOIN, GROUP BY operatorlari
- Jadval yaratish va o'zgartirishlar
- Indekslar va optimizatsiya
- Stored procedures va triggers
- Data normalization
